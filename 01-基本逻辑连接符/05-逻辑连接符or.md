
# 逻辑连接符 `or`

这一章节我们终于补全了三个符合我们习惯的逻辑操作符：与、或、非。

```follow
term prop or(prop p0, prop p1) { (p0 ∨ p1) }
```

```follow
axiom or.def.1(prop p0, prop p1) {
  |- imp(or(p0,p1), imp(not(p0),p1))
  |- imp(imp(not(p0),p1), or(p0,p1))
}
```

## 真值表 

| 名字 | 第一组值 | 第二组值 | 第三组值 | 第四组值 | 
| :---: | :---: | :---: | :---: | :---: |
| `p0` | 0 | 0 | 1 | 1 | 
| `¬p0` | 1 | 1 | 0 | 0 | 
| `p1` | 0 | 1 | 0 | 1 |
| `¬p0→p1` | 0 | 1 | 1 | 1 |
