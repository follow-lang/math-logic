
# 条件量词

## `exist*`（At Most One, 存在至多一个）

定义逻辑描述 **最多存在一个** 的量词。

```follow
term prop exist*(set s0, prop p0) { ∃ * ( s0 , p0 ) }
```

```follow
axiom exist*.def(set s0, prop p0, set s1) {
  |- iff(exist*(s0,p0), exist(s1, forall(s0,imp(p0,eq(c(s0),c(s1))))))
  diff (s1, s0) (s1, p0)
}
```
